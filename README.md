🧠 Hacker Wordlist Generator (Python GUI)

A fast and customizable wordlist generator tool built with Python and Tkinter.
This application generates large password wordlists based on a base keyword using random patterns, symbols, numbers, and common password structures.

⚠️ This tool is intended for educational purposes, cybersecurity practice, and ethical penetration testing only.

🚀 Features

Graphical User Interface (GUI)

Real-time wordlist generation

Multithreading for high-speed performance

Random password pattern creation

Automatic progress counter

Save generated wordlist to .txt file

Hacker-style terminal design

📦 Requirements

Python 3.x

Tkinter (usually included with Python)

No external libraries are required.

▶️ How to Download
Option 1 — Download ZIP

Click the green Code button on this repository

Select Download ZIP

Extract the project on your computer

Option 2 — Clone with Git
git clone https://github.com/YOUR_USERNAME/hacker-wordlist-generator.git

Then navigate into the folder:

cd hacker-wordlist-generator
⚙️ How to Run

Execute the script from your terminal:

python wordlist_generator.py

or on Linux / macOS:

python3 wordlist_generator.py
🖥️ How to Use

Enter a base word (example: john, company, dragon)

Click START to begin generating password combinations

The tool will continuously generate new entries

Click STOP & SAVE to stop generation and save the wordlist

Choose a location to export the .txt file

The generated wordlist can be used for:

Password security testing

Brute force simulations in lab environments

Ethical hacking practice

Cybersecurity research
